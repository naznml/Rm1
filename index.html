Официальная литература №1

Азимов Э.Г., Щукин А.Н. Новый словарь методических терминов и понятий

Структура книги сохраняется: предисловие, словарь, источники, алфавитный указатель, тематический указатель и приложение. Внутри разделов раскрываются понятия, нужные для КТ.

Структура книги №1

1. Предисловие

Назначение словаря, роль терминологии, связь методики с базисными дисциплинами.

2. Словарь

Методика, система обучения, подходы, принципы, технологии, речевая деятельность, контроль.

3. Источники

Научная и учебно-методическая база словаря.

4. Алфавитный указатель

Поиск терминов по алфавитному принципу.

5. Тематический указатель

Группировка терминов по содержательным областям методики.

6. Приложение

Дополнительная справочная часть.

Официальная литература №9

Новиков А.М., Новиков Д.А. Методология: словарь системы основных понятий

Книга используется для раскрытия методологической основы деятельности учителя: цель, задача, средство, результат, система, структура, анализ, проектирование, контроль, критерий, рефлексия, активность, способности и мышление.

Структура книги №9

1. Предисловие

Методология как учение об организации деятельности.

2. Словарь системы основных понятий

Деятельность, система, цель, задача, метод, анализ, проектирование, рефлексия.

3. Литература

Научная база словаря методологических понятий.

4. Справочный аппарат

Дополнительные сведения и поддержка работы с понятиями.
